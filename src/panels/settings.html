Ram Parametreleri.
Oyununuza ayrılan RAM'in tanımı, daha büyük bir miktar oyununuzun performansını artıracaktır..
Çok fazla miktar ram seçmekten kaçınmak gerektiğini unutmayın, aksi takdirde performans kaybına neden olabilir. autres applications.

- Maximum RAM : Oyununuza ayrılan toplam RAM miktarını gösterir.Doğru çalışabilmesi için PC'nizde bulunan toplam RAM miktarının %50'sini geçmemeniz önerilir. eğer bilgisayarınız 8GB'tan fazla RAM'e sahipse, 3GB ayırmanızı öneririz. Bilgisayarınız 4 GB veya daha az ise 2 GB'den fazla ayırma yapmayın..

- Minimum RAM : Oyuna başlarken ayrılan minimum RAM miktarını gösterir.Bu değer için 3GB'tan fazla koymamanız tavsiye edilir. Tipik değer 1 GB'dir.
Toplam Ram Miktarin 0 Seç RAM.
Toplam Ram Miktarin 0 Seç Kullanılabilir RAM.
